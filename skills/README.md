# Skills

Operational playbooks for validating and packaging Mechanize Schematics.

- `validate-schematic-contracts.md`: schema compliance for machine payloads.
- `validate-cog-compatibility.md`: semantic version and breaking-change checks for reusable cogs.
- `validate-platform-coverage.md`: required adapter and heartbeat coverage checks.
- `package-machine-artifacts.md`: deterministic packaging workflow for distribution.
